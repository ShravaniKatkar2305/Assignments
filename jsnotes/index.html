Introduction to JavaScript

JavaScript is a synchronous ,non blocking i/o scripting language used to make websites interactive. It can change content, respond to user actions, handle data, and add logic to web pages.

Synchronous

  • In synchronous execution, tasks run one after another in order.
  • Each line waits until the previous line completes.

Non-Blocking I/O

  • Non-blocking I/O means JavaScript does not wait for slow tasks like reading files, database, or API calls.
  • It continues running other code while I/O operations complete in the background.
  • Example: You order food (I/O task) and meanwhile talk with friends (other tasks continue).

History of JavaScript

JavaScript was created in 1995 by Brendan Eich at Netscape. It was originally made in just 10 days and was called “LiveScript”. Later it was renamed to JavaScript and became the most popular scripting language for the web.

Variable: Declaration, Definition & Initialization

Declaration: Creating a variable name.

Definition: Reserve memory for the variable.

Initialization: Giving first value to the variable.

let a;          // declaration
a = 10;         // initialization
let b = 20;     // declaration + initialization

JavaScript Hoisting

Hoisting means JavaScript moves the names of variables and functions to the top before executing the code. But remember → only the name is lifted, not the value.

Simple example: A teacher writes all students' names in the register first, but marks are filled later.

✔ How Hoisting Works?

  • var — Hoisted but its value becomes undefined.
  • let and const — Hoisted but not initialized. They stay in the TDZ (Temporal Dead Zone).
  • Function declarations — Fully hoisted. So they can be called before their actual line.

✔ Simple Example (var Hoisting)

console.log(a);  // output: undefined
var a = 10;

/* Behind the scene:
var a;        // hoisted
console.log(a);
a = 10;       
*/

✔ Example (let & const — TDZ)

console.log(b); // ❌ ReferenceError (TDZ)
let b = 20;
TDZ means: the variable exists but cannot be used yet because JavaScript says — "Wait! I don’t know its value yet!"

✔ Function Hoisting Example

// Works (function is hoisted)
console.log(add(2, 3)); // 5

function add(x, y) {
    return x + y;
}

✔ Function Expression (Not Fully Hoisted)

// ❌ Error (var is hoisted but value is undefined)
console.log(sub(3, 2));

var sub = function(x, y) {
    return x - y;
};

✔ Quick Summary

Keyword Hoisted? Value before declaration
var Yes undefined
let Yes TDZ (error)
const Yes TDZ (error)
function Yes (fully) Can be used before declaration

JavaScript Functions

A function is a block of code designed to perform a specific task. It runs only when it is called. Functions help make code reusable and cleaner.

Simple example: A function is like a machine. You give input → machine works → gives output.

✔ Why do we use Functions?

  • To reuse code
  • To organize code
  • To make code clean and readable
  • To avoid repeating the same logic
>

Functional Programming

Functional Programming is a way of writing programs using pure functions,that means we can write a function in another function It focuses on what to do instead of how to do it.

Simple Example: A calculator is a perfect functional machine — You give input → it gives output → it does not change anything around it.

✔ Key Features

  • writing programs using only functions
  • where a function takes some input and gives output and
  • does not change anything else.

Example1-pure function

        function square(n) 
        {
           return n * n;
        }

        console.log(square(5)); // always 25

Static vs Dynamic Programming Languages

Languages are divided based on when type checking happens: Before running the program or while running the program.

1️⃣ Static Languages (Type checked before execution)

These languages check errors at compile-time. Variables must have a fixed type. 

// Example (Java - Static)
int x = 10;
x = "Hello"; // ❌ Error (string not allowed)
  • Fewer runtime errors
  • Fast performance
  • More strict

✔ Examples of Static Languages

  • Java
  • C
  • C++
  • Go
  • Rust

2️⃣ Dynamic Languages (Type checked at runtime)

These languages allow variables to change their type while running the program. Type errors appear while executing code. 

// Example (JavaScript - Dynamic)
let a = 10;
a = "Hello"; // ✔ Allowed
  • More flexible
  • Easy to write
  • Faster development

✔ Examples of Dynamic Languages

  • JavaScript
  • Python
  • Ruby
  • PHP
  • Perl

✔ Quick Summary

Language Type Type Checking Flexibility Examples
Static Before running (Compile-time) Strict Java, C, C++, Rust
Dynamic While running (Runtime) Flexible JavaScript, Python, Ruby

JavaScript Array (Simple Explanation)

An array is a special variable in JavaScript that stores multiple values in one place. Example: You can store many names, numbers, or colors together.

Simple Example: A box that contains many items. That box = Array Items inside it = Values

✔ How to Create an Array?

let numbers = [10, 20, 30, 40];     // array of numbers
let names = ["shravni", "Siddhi", "vedika","yogita","pragati"]; // array of strings
let mixed = [10, "hello", true];    // mixed values

✔ Accessing Array Values (Using Index)

Index starts from 0.

let fruits = ["Apple", "Banana", "Mango"];

console.log(fruits[0]); // Apple
console.log(fruits[1]); // Banana
console.log(fruits[2]); // Mango

✔ Changing Array Values

let colors = ["Red", "Blue", "Green"];

colors[1] = "Yellow";   // change Blue to Yellow

console.log(colors);
// Output: ["Red", "Yellow", "Green"]

✔ push() — Add Value at the End

let animals = ["Dog", "Cat"];

animals.push("Cow"); 
console.log(animals);

// Output: ["Dog", "Cat", "Cow"]

✔ pop() — Remove Last Value

let items = ["Pen", "Pencil", "Eraser"];

items.pop();  
console.log(items);

// Output: ["Pen", "Pencil"]

✔ unshift() — Add Value at the Start

let cars = ["BMW", "Audi"];

cars.unshift("Mercedes");
console.log(cars);

// Output: ["Mercedes", "BMW", "Audi"]

✔ shift() — Remove First Value

let games = ["Cricket", "Football", "Hockey"];

games.shift();
console.log(games);

// Output: ["Football", "Hockey"]

✔ length — Total Items in Array

let cities = ["Mumbai", "Pune", "Delhi", "Nagpur"];

console.log(cities.length);  
// Output: 4

✔ Looping Through an Array

let students = ["Shravani", "Siddhi", "vedika"];

for (let i = 0; i < students.length; i++) {
    console.log(students[i]);
}

/*
Output:
Shravani
Siddhi
vedika
*/
