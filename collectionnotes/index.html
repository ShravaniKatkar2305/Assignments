What is collection?

The Collection is a ready-to-use implementation of data structure provided by Java.
Collection is a set of objects stored in a single variable.
The Object class is the parent of every class.
Primitive data types are not supported in collection.

Key Features:

  • Root interface of Java Collections Framework.
  • Provides common methods like add(), remove(), size(), clear(), isEmpty().
  • Supports iteration using Iterator.
  • Implemented by various subinterfaces like List, Set, and Queue.
  • Does not allow primitive types directly; use wrapper classes (Integer, Double, etc.).

List Interface - Java

The List interface represents an ordered collection of elements where duplicates are allowed.

Types of List:

  • ArrayList: Resizable array. Fast access, slow insert/delete in middle.
  • LinkedList: Doubly-linked list. Fast insert/delete, slow random access.
  • Vector: Thread-safe resizable array (legacy).
  • Stack: LIFO structure extending Vector.
